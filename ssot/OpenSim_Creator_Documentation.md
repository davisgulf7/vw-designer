# OpenSim Creator Documentation[#](https://docs.opensimcreator.com/#opensim-creator-documentation)

OpenSim Creator is a UI for building OpenSim models. These pages are the official documentation for OpenSim Creator, where you can find installation instructions, tutorials that cover specific workflows/features, the development guide, and other related links.
