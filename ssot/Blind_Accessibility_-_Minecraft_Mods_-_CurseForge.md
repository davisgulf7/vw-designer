# Blind Accessibility (Minecraft Access)

Minecraft Access is a [Minecraft](https://www.curseforge.com/linkout?remoteUrl=https%253a%252f%252fminecraft.net%252f) mod that specifically helps visually impaired players play Minecraft. It is an integration and replacement for [a series of previous mods](https://github.com/accessible-minecraft). This mod primarily borrows the help of a screen reader to describe (narrate) the game interface, and incorporates sound cues to provide orientation perception in this 3D world. Currently, this mod [has enough features](https://www.curseforge.com/linkout?remoteUrl=https%253a%252f%252fdocs.mcaccess.org%252ffaq%2523is-the-mod-enough-to-play-the-game-normally) to help visually impaired players play the game normally.

This mod has available versions for:

Game version  1.19.3 ,  1.20.1 ,  1.20.4 ,  1.20.6 ,  1.21.1 ,  1.21.4 .

On the [Fabric](https://www.curseforge.com/linkout?remoteUrl=https%253a%252f%252ffabricmc.net%252fuse%252finstaller%252f) and [NeoForge](https://www.curseforge.com/linkout?remoteUrl=https%253a%252f%252fneoforged.net%252f) mod loaders

On Windows, macOS and Linux operating systems

Whenever the code is updated, the corresponding latest snapshot version will be published in the [Playability Discord server](https://discord.gg/yQjjsDqWQX). After gathering enough changes and feedback for making sure there is no serious bug, a tagged version will be released on [GitHub](https://github.com/khanshoaib3/minecraft-access/releases), [Modrinth](https://www.curseforge.com/linkout?remoteUrl=https%253a%252f%252fmodrinth.com%252fmod%252fminecraft-access%252fversions) and [CurseForge](https://legacy.curseforge.com/minecraft/mc-mods/blind-accessibility/files).
