# Open LMS WORK Managed Hosting Release Schedule

**Jeremy Schweitzer**

Updated 10 months ago

Follow Not yet followed by anyone

Customers hosted in Managed Hosting have the option to set their own upgrade date for major releases. This is aligned to changes in the underlying major Moodle™ version. For example, upgrading from 1.11 to 2.1 or from 2.4 to 3.0 are major releases. Minor releases are handled during maintenance windows and are not available for scheduling. Examples of these are upgrades from WORK 3.0 to 3.1. Sandboxes will be upgraded shortly after each release and cannot be individually scheduled.

As of January 2025, the current WORK release is 4.0. The 4.x releases are based on Moodle™ 4.5. The 3.x releases were based on Moodle™ 4.3, while WORK 2.x was based on the 4.1 branches.

For customers hosted in Managed Hosting, they can choose their own date, but must upgrade to a version of WORK 4.x before June 1, 2025.

Since customers can set their specific upgrade date, the dates below are the earliest available dates. This does not guarantee that any specific date will be available. Scheduling can be conducted via a support ticket.

We update this page with new information when it's available.

Production Upgrades can begin: Tuesday, **January 21, 2025**

Upgrades can begin **March 19, 2025**

**Open LMS WORK 4.2 (Moodle™ 4.5 Version)**

**Open LMS WORK 4.3 (Moodle™ 4.5 Version)**
