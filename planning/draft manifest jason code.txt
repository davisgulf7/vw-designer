You want three outputs, not one:

1. manifest.json
   The master structured knowledge layer.

2. dashboard_index.json
   A lighter file for browsing/filtering sources quickly.

3. ssot_seed_map.json
   A file that maps source IDs to the SSOT documents they should feed.
